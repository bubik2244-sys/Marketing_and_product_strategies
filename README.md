# Customer Shopping Behavior Analysis

This project analyzes customer shopping behavior data using Python, Pandas, SQL, Excel, Power BI, and Jupyter Notebook. The main goal is to clean the dataset, handle missing values, create useful new columns, analyze customer patterns, and prepare insights for dashboard visualization.

## Tools Used

* Python
* Pandas
* Jupyter Notebook
* Excel
* SQL
* Power BI
* GitHub

## Main Work

* Loaded CSV data using Pandas
* Checked and handled missing values
* Filled null values using median
* Created age group categories
* Converted purchase frequency into days
* Removed unnecessary columns
* Performed basic exploratory data analysis
* Used SQL concepts for data querying and analysis
* Planned Power BI dashboard for visualization

## Key Analysis Areas

* Customer age group analysis
* Purchase amount analysis
* Category-wise shopping behavior
* Review rating analysis
* Purchase frequency analysis
* Discount and promo code impact

## Purpose

This project is part of my data analysis learning journey and portfolio preparation. It shows my basic skills in data cleaning, analysis, SQL thinking, and dashboard preparation.
